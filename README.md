# 🎵 Spotify User Analysis

An interactive Microsoft Excel dashboard that analyzes Spotify user behavior, subscription preferences, listening habits, device usage, and country-wise user distribution. This project demonstrates data cleaning, Pivot Tables, Pivot Charts, KPI cards, slicers, and dashboard creation using Microsoft Excel.

---

## 📌 Project Overview

This project explores Spotify user data to uncover insights into user engagement and listening behavior. The dashboard provides an interactive way to analyze subscription plans, listening time, device preferences, and user distribution across countries.

## 📁 Dataset Information

The dataset contains information for **8,000 Spotify users**.

### Dataset Features

| Column                | Description                     |
| --------------------- | ------------------------------- |
| user_id               | Unique identifier for each user |
| gender                | User gender                     |
| age                   | User age                        |
| country               | User country                    |
| subscription_type     | Subscription plan               |
| listening_time_in_min | Daily listening time (minutes)  |
| songs_played_per_day  | Songs played each day           |
| skip_rate             | Percentage of skipped songs     |
| device_type           | Device used for listening       |
| ads_listened_per_week | Weekly ads listened to          |
| offline_listening     | Offline listening status        |
| is_churned            | User churn indicator            |

- Desktop is the most preferred listening device.
- User distribution is fairly balanced across all countries.
- Users spend an average of **154 minutes** listening to Spotify.
- More than **400,000 songs** are streamed daily by all users combined.
